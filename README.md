📝 Todo - Modern Todo App

A modern Todo Application built using HTML, CSS, and JavaScript. Todo helps users organize daily tasks efficiently with a clean interface, smooth animations, and an intuitive user experience.

✨ Features

- ➕ Add new tasks instantly
- 📅 Assign dates to tasks
- 🗑️ Delete completed tasks
- 🎨 Clean and modern user interface
- ⚡ Smooth button animations and hover effects
- 📱 Responsive design for different screen sizes
- 🚀 Fast and lightweight

🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

📸 Preview

Todo provides a simple and elegant task management experience designed to improve productivity and organization.

🚀 Getting Started

1. Clone the repository:
   
   git clone https://github.com/your-username/Todo-App.git

2. Open the project folder.

3. Run "index.html" in your browser.
